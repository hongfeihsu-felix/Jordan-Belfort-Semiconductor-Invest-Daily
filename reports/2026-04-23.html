🏢 细分领域动态

SK海力士 vs 三星：HBM军备竞赛加速

核心变化： SK海力士4/20宣布量产SOCAMM2（192GB），三星同时将大量DRAM产能转产HBM（对特斯拉DRAM交货量+400% YoY），两大韩国存储巨头同步向AI内存倾斜。

具体数据： TrendForce数据显示Q1 DRAM合约价暴涨90-95%QoQ，NAND Flash合约价+55-60%QoQ；HBM已消耗23%的DRAM晶圆产能；HBM供应商毛利率达60-70%历史高位。

竞争影响： 三星将HBM产能优先分配给英伟达，导致标准DRAM现货持续紧张——16Gb DDR5现货价格过去一年飙涨598%。手机OEM（苹果/三星）被迫重新谈判合约，利润率承压。SK海力士凭借SOCAMM2在AI服务器内存市场建立先发优势，对三星形成正面挑战。

台积电：AI芯片制造垄断地位持续强化

核心变化： 台积电Q1净利润$18.2B（历史最高），毛利率66.2%；全年营收增长预期从"接近30%"上调至">30%"，CapEx维持$52-56B区间上沿。

具体数据： 2026年AI基础设施总支出预计达$660B；TSMC Q2营收指引$39-40.2B（+32% YoY）；英伟达、苹果、AMD、博通等大客户对CoWoS封装需求远超产能。

竞争影响： ASML和台积电双双上调全年预期，被视为AI spending boom未减的明确信号。博通/AMD/英伟达均依赖台积电先进制程，台积电议价能力空前强化——高盛预计其毛利率将持续维持在60%以上直到N2量产爬坡完成。

高通 vs 联发科：印度市场冰火两重天

核心变化： Q1 2026印度手机出货量创六年新低，约1/3新机发布推迟至Q1以消化成本压力；但Counterpoint数据显示联发科以49%出货量份额守住印度市场第一，高通则在高端安卓（>$400）占据>50%份额。

具体数据： 80款以上机型均价上涨约15%；入门DRAM BOM上涨40-50%，联发科中低端SoC面临成本转嫁困境；高通X2代旗舰SoC（Hexagon NPU）AI算力达80 TOPS。

竞争影响： 联发科出货量大但单价受压——天玑系列毛利率低于骁龙；高通凭借高端市场主导地位（小米14 Ultra、OPPO Find X8等旗舰均用骁龙）在ASP和利润率上占优。成熟制程代工吃紧对两家均构成供给侧约束，但高通因溢价能力强转嫁能力更佳。

🔗 传导链追踪

存储涨价传导链（当前：上行后段，距周期顶部约2季度）

AI服务器需求爆发（2025Q3起） → HBM产能满载+转产加速（2025Q4） → DRAM合约价+90-95% QoQ（2026Q1，历史最大涨幅） → 入门机BOM上涨40-50%（2026Q1-Q2） → 印度/新兴市场出货量下滑（2026Q1，已验证） → 传音/小米部分机型定价冻结 → 存储厂开始评估扩产（2026H2信号窗口） → 供需平衡改善（2027Q1预期）

AI芯片需求链（当前：加速上行，距离顶部尚远）

ChatGPT引爆AI应用（2023年初） → 英伟达H100/H200需求爆发（2023-2024） → 台积电CoWoS封装成瓶颈（2024-2026，持续） → 台积电先进制程产能满载（2026全年） → AI基础设施CapEx $660B（2026） → 博通/AMD/苹果追加订单（2026Q1） → TSMC全年增长>30%（已实现）

手机SoC竞争链（当前：高端强势、中低端承压）

先进制程产能 → 台积电（稀缺） → 旗舰SoC（骁龙8/X2代） → 高通主导（>50%高端份额） → 旗舰手机涨价但需求刚性 → 中低端SoC（天玑7000系） → 成熟制程配给优先权低 → 成本上涨无法完全转嫁 → 联发科出货量份额稳但毛利率受压 → 传音等入门机厂商盈利大幅下滑
